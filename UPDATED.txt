To use the QR code generator:
1. Click the QR code icon in the header (next to the language selector)
2. A modal will appear showing the QR code for the current page URL
3. The QR code can be downloaded by right-clicking and saving the image

To add product pictures:
1. Place image files in the assets/products/ directory
2. Update the image paths in data/menu.json accordingly
3. Recommended format: JPG or WebP, optimized for web
