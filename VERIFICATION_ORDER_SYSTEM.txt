🔧 HOW IT WORKS:
1. User clicks on a product card to open modal
2. Modal displays product details with "Add to order" button
3. When button is clicked:
   - Currently displayed product ID is retrieved from modal title
   - Product is saved to localStorage (or quantity incremented if already exists)
   - Toast confirmation appears: "Produit ajouté à la commande!"
4. Order data remains in localStorage until manually cleared
5. Can be retrieved later by reading localStorage.getItem('menuOrders')

📝 USAGE EXAMPLE:
After adding items, orders can be retrieved with:
const orders = JSON.parse(localStorage.getItem('menuOrders') || '[]');
console.log(orders); // Array of ordered products with quantities
