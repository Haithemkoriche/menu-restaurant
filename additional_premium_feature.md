# Additional Premium Features



## WhatsApp Order System

Implement a "Commander via WhatsApp" button.

When clicked:

The application must automatically generate a professional order message.

Example:

Bonjour,

Je souhaite passer la commande suivante :

Table N°5

* 2 Tacos Poulet
* 1 Coca-Cola
* 1 Pizza Margherita

Total : 3 150 DA

Merci.

The application must automatically open WhatsApp using:

https://wa.me/xxxxxxxxxxx?text=generated_message

The restaurant phone number must be configurable from a settings file.

No backend server is allowed.

Everything must work directly from the frontend.
